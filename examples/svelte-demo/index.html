‚ö†Ô∏è Svelte Components Require Build Step

Unlike Vue and React which can run in the browser via CDN, Svelte components need to be compiled. This demo shows you how to use the components in a real Svelte/SvelteKit project.

Why? Svelte compiles components at build time rather than runtime, which makes the final bundle smaller and faster, but means you can't load them directly in the browser like Vue/React.

npm install @git-stats-components/svelte

üöÄ SvelteKit Usage

<!-- src/routes/+page.svelte -->
<script lang="ts">
  import { ContributionGraph, StatsBreakdown } from '@git-stats-components/svelte'
  import type { ExperienceEntry } from '@git-stats-components/svelte'
  
  let colorScheme: 'green' | 'blue' | 'purple' | 'orange' = 'green'
  
  const experienceData: ExperienceEntry[] = [
    {
      startDate: '2020-01-01',
      endDate: null,
      skills: ['JavaScript', 'Svelte', 'SvelteKit', 'TypeScript']
    }
  ]
  
  function handleDayClick(event: CustomEvent) {
    const { date, count } = event.detail
    console.log(`Clicked: ${date} with ${count} contributions`)
  }
</script>

<div class="container">
  <h2>My Contributions</h2>
  
  <ContributionGraph
    dataUrl="/data/git-stats.json"
    {colorScheme}
    showSettings={true}
    on:dayClick={handleDayClick}
  />
  
  <StatsBreakdown
    dataUrl="/data/git-stats.json"
    {experienceData}
    showCustomStat={true}
  />
</div>

<style>
  .container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 40px 20px;
  }
</style>

‚öôÔ∏è Props & Events

ContributionGraph

Props:
  dataUrl: string = '/data/git-stats.json'
  profileIndex: number = 0
  colorScheme: 'green' | 'blue' | 'purple' | 'orange' = 'green'
  showSettings: boolean = true
  cacheTTL: number = 24 * 60 * 60 * 1000

Events:
  on:dayClick - { date: string, count: number }
  on:colorSchemeChange - ColorScheme

Slots:
  settings-icon - Custom settings icon

StatsBreakdown

Props:
  dataUrl: string = '/data/git-stats.json'
  profileIndexes: number[] = []
  experienceData: ExperienceEntry[] = []
  showCustomStat: boolean = true
  customStatCalculator: Function | null = null
  cacheTTL: number = 24 * 60 * 60 * 1000

Slots:
  icon-experience - Custom experience icon (default: ‚è±Ô∏è)
  icon-projects - Custom projects icon (default: üì¶)
  icon-commits - Custom commits icon (default: üíª)
  icon-custom - Custom stat icon (default: ‚òï)
  custom-stat-label - Custom stat label (default: Coffee Consumed)
  default - Additional content at bottom

üí° Why Use Svelte?

Smaller bundles: Svelte compiles away, resulting in tiny bundle sizes
Better performance: No virtual DOM overhead
Simpler syntax: Less boilerplate than Vue/React
Reactive by default: Just assign variables, no useState or ref
